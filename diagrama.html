Framework Interbank · Riesgo de tasa (IRRBB) · Tarjetas de crédito

Del dato transaccional al insumo regulatorio para ∆EVE

Flujo de las tres etapas del modelo: segmentación conductual, utilización de línea y velocidad de amortización — y cómo se combinan para alimentar la medición de riesgo de tasa bajo SRP31.
Datos crudos (input)
Procesamiento (modelo)
Output del modelo
Insumo regulatorio / decisión
Inputs
Base transaccional
Datos cliente–mes
Límite de crédito, saldo facturado, monto pagado, por cliente y fecha.
Base macroeconómica
Series mensuales
Confianza del consumidor, masa salarial, spread de tasas, desempleo, estacionalidad.
Configuración
Parámetros del modelo
Ventana de observación (6 meses), umbral de tolerancia de pago (95%).
Etapa 1
Segmentación conductual
Rolling window 6 meses
Clasifica cada cliente-mes como pagador o no, y aplica una ventana móvil para etiquetar el comportamiento dominante.
Output etapa 1
Segmento_Conductual
Totalero vs. Revolvente — determina qué saldo entra al resto del pipeline.
Etapa 2
Modelo de utilización
SARIMAX (1,1,1)
Predice qué % del cupo disponible usa la cartera revolvente cada mes, en función de macro y estacionalidad.
Output etapa 2
Tasa_Utilizacion
% del límite de crédito en uso — insumo de exposición para el cálculo de flujos futuros.
Etapa 3
Modelo de amortización
Logit + Regresión OLS
Predice la velocidad de pago mensual (TPR) en función del spread de tasas y el desempleo rezagado.
Output etapa 3
TPR_Predicho
Tasa de pago mensual esperada — la "velocidad" a la que el saldo revolvente sale del balance.
Cierre
Sanity check
Curva de decay implícita
Vida media no paramétrica derivada del TPR mensual, usada para contrastar contra el decay estilo NMD.
Sₜ = S₀ · ∏(1 − TPRᵢ)
Insumo regulatorio
Proyección de flujos y ∆EVE
Slotting temporal, descuento y cálculo de impacto de shocks de tasas bajo los seis escenarios de SRP31.
Nota metodológica: las tres etapas comparten como insumo la base macroeconómica; las líneas punteadas indican esa dependencia transversal, no un flujo secuencial adicional.
